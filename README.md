# PetConnect – Aplicación Móvil para Adopción de Mascotas

## Descripción del Proyecto

PetConnect es una aplicación móvil nativa para Android orientada a facilitar el proceso de adopción de mascotas mediante la conexión entre refugios de animales y personas interesadas en adoptar.

La aplicación permitirá visualizar mascotas disponibles para adopción, gestionar publicaciones realizadas por refugios y ofrecer información organizada sobre cada animal.

Este proyecto busca brindar una solución tecnológica sencilla y funcional frente a la problemática del abandono animal y la falta de plataformas organizadas para procesos de adopción.

---

# Problemática

Actualmente, muchos refugios y organizaciones de rescate animal utilizan principalmente redes sociales para publicar mascotas disponibles para adopción. Esto genera desorganización, pérdida de información y poca visibilidad de las publicaciones.

Asimismo, las personas interesadas en adoptar deben buscar información en distintas plataformas, dificultando encontrar mascotas que se adapten a sus preferencias.

Por ello, PetConnect propone centralizar la información mediante una aplicación móvil moderna y fácil de utilizar.

---

# Objetivo General

Desarrollar una aplicación móvil para Android que permita visualizar mascotas disponibles para adopción y facilite la conexión entre refugios y personas interesadas en adoptar.

---

# Objetivos Específicos

- Diseñar una interfaz amigable e intuitiva.
- Implementar un sistema de registro e inicio de sesión.
- Permitir la publicación de mascotas disponibles para adopción.
- Desarrollar una vista detallada de mascotas.
- Implementar una sección de mascotas favoritas.
- Utilizar Firebase para almacenamiento y autenticación.
